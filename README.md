## Etapas para executar a aplicação 

- Clone o repositório.
```
https://github.com/robinhorios/customer_success_balancing
```
- Instale a GEM minitest
```
gem install minitest
```
- Execute o comando abaixo
```
ruby customer_success_balancing.rb
```
